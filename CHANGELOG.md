# Changelog

All notable changes to this project are documented in this file.

## 0.3.0 - 2026-04-18

### Added

- Rename product to PromptManager
- Side-panel toast notifications for copy, save, delete actions
- Non-blocking delete flow with undo support
- Click prompt card to copy content and load editor
- Refined hover/active interaction feedback and copied pulse effect
- Improved scrollbar styling and visual consistency in side panel
- Open-source and store-readiness docs: LICENSE, PRIVACY, CONTRIBUTING, SECURITY, release checklist

### Changed

- Upgraded UI consistency for category and prompt panes
- Updated branding and README instructions
