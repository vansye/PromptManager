# Contributing

感谢你对 PromptManager 的关注。

## 开发环境

```bash
npm install
npm run dev
```

## 提交规范

- 提交前请先运行构建检查：

```bash
npm run build
```

- 提交信息建议格式：
  - `feat: ...`
  - `fix: ...`
  - `docs: ...`
  - `refactor: ...`

## Pull Request 要求

- 说明变更目的与影响范围
- 若涉及 UI，请附截图
- 如修复 bug，请说明复现步骤与修复点

## 行为准则

请保持礼貌、专业、建设性的沟通。
